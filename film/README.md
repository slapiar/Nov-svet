# Nov-svet: Film

> Filmová časť projektu Nov-svet

## Štruktúra

```
film/
├── scenar/        # Scenár a dramaturgické materiály
│   └── scenar.md
```

## Produkcia

Film zachytáva príbehy ľudí, ktorí prešli transformáciou medziľudských vzťahov
vďaka princípom projektu Nov-svet.

## Formát

- Dĺžka: 90 minút
- Žáner: Dokumentárno-hrané dielo
- Jazyk: Slovenský, s možnosťou titulkov
