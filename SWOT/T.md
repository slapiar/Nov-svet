# T – Threats / Hrozby

Tento dokument je pracovný register hrozieb zdroja SWOT projektu Nový svet.

## Účel

Zachytáva vonkajšie riziká, prekážky, tlaky, zlyhania prostredia, konkurenčné alebo systémové ohrozenia a udalosti, ktoré môžu ohroziť rozvoj projektu.

## Pravidlo zápisu

Každý záznam má byť formulovaný tak, aby bolo jasné:

- čo je hrozba,
- odkiaľ prichádza,
- aký má možný dopad,
- či ide o technické, procesné, personálne, právne, vzťahové alebo trhové riziko,
- akú príležitosť, obranný proces alebo WHAT IF scenár môže spustiť.

## Pracovná šablóna záznamu

```text
ID:
Názov:
Popis:
Zdroj hrozby:

Napojené moduly: AS_IS / TO_BE / WHAT_IF / ZEPHYR
Možná reakcia:
Nadväzný proces:
Stav: draft / active / reviewed / archived
```
