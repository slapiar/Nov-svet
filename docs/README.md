## Rýchly štart

```bash
# Klonovanie repozitára
git clone https://github.com/slapiar/Nov-svet.git
cd Nov-svet

# Inštalácia závislostí
npm install

# Spustenie webovej aplikácie
npm run dev

# Spustenie testov
npm test
```

## Moduly

| Modul   | Popis                              | Stav           |
|---------|------------------------------------|----------------|
| `web`   | Online systém (Node.js/Express)    | V vývoji       |
| `book`  | Knižná publikácia (Markdown)       | V vývoji       |
| `game`  | Počítačová hra                     | V vývoji       |
| `film`  | Filmová produkcia                  | V príprave     |
