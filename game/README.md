# Nov-svet: Hra

> Počítačová hra o vývoji medziľudských vzťahov

## Spustenie

```bash
npm install
npm start
```

## Vývoj

```bash
npm run dev   # Sledovanie zmien
npm test      # Spustenie testov
npm run lint  # Kontrola kódu
```

## Štruktúra

```
game/
├── src/
│   ├── main.js    # Vstupný bod
│   └── hra.js     # Herné jadro
├── assets/        # Herné prostriedky (obrázky, zvuky)
├── tests/         # Testy
└── package.json
```

## Herný zámer

Hráč naviguje sociálnymi situáciami, buduje vzťahy a rozvíja emocionálne
kompetencie. Hra simuluje rôzne typy medziľudských interakcií.
